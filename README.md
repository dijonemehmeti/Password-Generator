# Password-Generator

A simple interactive web application that generates secure passwords based on user-selected criteria.  
This project was created to practice front-end development using **HTML**, **CSS**, and **JavaScript**, focusing on DOM manipulation, form handling, and logic-based password generation.

---

## Features
- Generate a random password instantly
- Choose password **length**
- Toggle options for including:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Copy the generated password to clipboard
- Fully responsive layout with clean UI
- Lightweight and runs entirely client-side (no backend)
